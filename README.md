# Leena Magar - Personal Portfolio 🎨

A modern, lightweight, and highly responsive personal portfolio website designed for Leena Magar, an IT student and Web Developer.

The design utilizes a clean, bright **"Retro UX"** aesthetic, featuring pastel purples, deep indigo text, mint green accents, and elegant serif typography to stand out from standard developer portfolios.

## ✨ Features

- **Modern Aesthetic:** A bright, playful, yet highly professional design inspired by high-end UX design portfolios.
- **Responsive Layout:** Fully optimized for desktops, tablets, and mobile devices.
- **Split-Layout Design:** Engaging side-by-side content sections for improved readability.
- **Zero Dependencies:** Built entirely with vanilla web technologies for lightning-fast loading speeds.
- **Scroll Animations:** Smooth fade-in and slide-up reveal animations as you scroll down the page.

## 🛠️ Technologies Used

- **HTML5:** Semantic and accessible document structure.
- **CSS3:** Custom styling, CSS Grid/Flexbox layouts, and CSS variables for easy theme management.
- **JavaScript (Vanilla):** Intersection Observer API for scroll animations.
- **Google Fonts:** Featuring `Playfair Display` (serif) and `Plus Jakarta Sans` (sans-serif).
- **Phosphor Icons:** Clean and lightweight iconography.

## 🚀 How to Run Locally

Because this project is built with static files, running it is incredibly simple.

1. Clone or download the repository to your local machine.
2. Open the project folder (`Leena_Portfolio`).
3. You can run the project in two ways:
   - **Option A:** Double-click the `index.html` file to open it directly in your browser.
   - **Option B (Recommended):** Use a local development server like VS Code's "Live Server" extension, or run `npx serve` in your terminal to view it at `http://localhost:3000`.
